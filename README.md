# 💜 MyWallet — Personal Finance Manager

MyWallet is a personal finance web application built with Flask and SQLite, 
designed for students, freelancers, and young professionals who want better 
control over their money.

## ✨ Features

- 📊 **Dashboard** — daily spending limit, today's expenses, smart suggestions
- 💰 **Income Tracking** — pocket money, salary, freelance, scholarships & more
- 🧾 **Expense Management** — add, edit, delete, search & filter expenses
- 🏷️ **Categories** — 10 default categories + create your own
- 💼 **Budget Planner** — set monthly budgets per category with warnings
- 🐷 **Savings Goals** — track progress towards financial goals
- 📈 **Analytics** — spending charts, monthly comparisons & AI budget recommendations

## 🛠️ Tech Stack

- **Backend** — Python, Flask, SQLAlchemy
- **Database** — SQLite
- **Frontend** — HTML, Jinja2, Bootstrap 5, Chart.js
- **Auth** — Flask-Login

## 🚀 Getting Started

```bash
pip install -r requirements.txt
python app.py
```

Then open `http://localhost:5000` in your browser.

# Technology Stack

# Project Name

MyWallet

---

# 1. Overview

This document describes the technologies used in the MyWallet project and explains why each technology has been selected.

The goal is to choose technologies that are simple, scalable, beginner-friendly, and suitable for future upgrades.

---

# 2. Programming Language

## Python

Purpose

* Main application logic
* Business logic
* Data processing

Reason

Python is easy to read, beginner-friendly, and provides built-in libraries for database connectivity and data analysis.

Future Uses

* Analytics
* AI Budget Recommendations
* Machine Learning

---

# 3. Database

## SQLite

Purpose

Store all application data locally.

Reason

* Lightweight
* No installation required
* Perfect for desktop applications
* Fast
* Built into Python

Future Upgrade

PostgreSQL or MySQL
